## Description — English

Serial Air turns your phone into a wireless serial monitor for Arduino, ESP8266, ESP32, and other WiFi-enabled microcontrollers.

Stop tethering to your PC with USB cables. Debug your IoT devices from anywhere on your WiFi network — even when they're mounted on ceilings, embedded in enclosures, or deployed outdoors.

▸ HOW IT WORKS
1. Add the free WirelessSerial library to your Arduino sketch (2 lines of code)
2. Upload to your ESP8266 or ESP32
3. Open Serial Air — your device appears automatically
4. Tap to connect and start monitoring

▸ KEY FEATURES
• Auto-Discovery — Devices found automatically via Bonjour/mDNS
• Real-time Monitoring — Live serial output with millisecond timestamps
• Send Commands — Bidirectional communication with your device
• Search & Filter — Find specific log entries instantly
• Export Logs — Save and share log files
• Auto-Reconnect — Seamless recovery from network interruptions
• Manual Connection — Connect via IP address and port
• Dark Terminal UI — Easy on the eyes during long debug sessions

▸ WORKS WITH
• ESP8266 (NodeMCU, Wemos D1 Mini, etc.)
• ESP32 (DevKit, TTGO, Heltec, etc.)
• Any WiFi microcontroller with TCP support

▸ BUILT FOR DEVELOPERS
• Minimal memory footprint — WirelessSerial uses < 2KB RAM on ESP8266
• Standard Telnet protocol (port 23) — also works with any TCP client
• Open-source Arduino library on GitHub

▸ PRICING
• 7-day free trial — full features, no restrictions
• $1.99 one-time purchase to unlock permanently
• No subscriptions, no ads, no data collection

▸ GETTING STARTED
The app includes a built-in code generator: enter your WiFi credentials, select your board, and copy the ready-to-upload test sketch. No documentation needed.

Arduino Library: github.com/matdonaruno/serial-air

## Description — Japanese (日本語)

Serial Airは、Arduino・ESP8266・ESP32のシリアル出力をスマートフォンでワイヤレスに確認できるアプリです。

USBケーブルとPCから解放されましょう。天井裏やケースの中、屋外に設置されたデバイスも、WiFi経由でリアルタイムにデバッグできます。

▸ 使い方
1. 無料のWirelessSerialライブラリをArduinoスケッチに追加（たった2行）
2. ESP8266/ESP32に書き込み
3. Serial Airを開く → デバイスが自動で見つかります
4. タップして接続、モニタリング開始

▸ 主な機能
• 自動検出 — Bonjour/mDNSでデバイスを自動発見
• リアルタイム表示 — ミリ秒タイムスタンプ付きシリアル出力
• コマンド送信 — デバイスとの双方向通信
• 検索・フィルター — ログを瞬時に絞り込み
• ログ書き出し — テキストファイルで保存・共有
• 自動再接続 — ネットワーク切断から自動復帰
• 手動接続 — IPアドレスとポートで直接接続
• ダークUI — 長時間のデバッグでも目に優しい

▸ 対応デバイス
• ESP8266 (NodeMCU, Wemos D1 Mini 等)
• ESP32 (DevKit, TTGO, Heltec 等)
• TCP対応のWiFiマイコン全般

▸ 料金
• 7日間無料トライアル — 全機能制限なし
• $1.99（300円）の買い切りで永久使用
• サブスクなし・広告なし・データ収集なし

▸ はじめ方
アプリ内のコード生成機能で、WiFi情報を入力するだけでテスト用スケッチが完成。コピーしてArduino IDEに貼り付け、書き込むだけです。
