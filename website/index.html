Setup

Up and running in 2 minutes

1

Add the Library

Add 2 lines to your Arduino sketch. That's it.

#include <WirelessSerial.h>
WirelessSerial ws;
DualPrint* output;

void setup() {
  ws.begin();
  output = ws.mirror(Serial);
}
2

Upload & Power On

Flash to your ESP8266 or ESP32. Connect to WiFi.

void loop() {
  ws.handle();
  output->println("Hello!");

}
